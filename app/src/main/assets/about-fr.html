À Propos

Ce quiz a été développé par Samuel Walladge pour Assignment 2, HIT238.

Le code source est disponible sur GitHub.

Comment utiliser

Voilà quelques questions pour évaluer vos connaissances des capitales de la monde. Pour commencer, appuyer sur le bouton « Commencer le Quiz ! ». Chaque question a 5 options possibles. On peut naviguer entre les questions et on peut appuyer sur « Soumettre les réponses » quand on est prêt à soumettre. Après soumettre, on est presenté avec un écran des resultats. Le lire attentivement, parce que on doit reprendre le quiz pour le voir encore !

Fonctionalités

  • Traduit en Anglais et Français.
  • Un selection des questions difficiles.
  • Les réponses sont sauvées automatiquement pour recommencer plus tard.
